---Mathematical Expression Evaluator---
[School project]

Evaluates expressions via commandline arguments; 
example: "a+b-c d*e f/g".

Different expressions are seperated by spaces.

Accepts the following operators: + - * /
(Designed to easily implement additional operators)

Takes arguments as strings, tokenizes them, and then evaluates the expressions they
form based on the types of tokens provided.
